# Inline

Онлайн-очередь с тайм-слотами. Организатор создаёт очередь, участники присоединяются по QR-коду или 4-значному коду.

## Возможности

- Создание очередей с настраиваемыми временными слотами
- Присоединение по QR-коду или 4-значному коду
- Автоматическое распределение участников по слотам
- Запросы на смену слота или позиции (с одобрения организатора)
- Кнопка "Задерживаюсь" для участников
- Автоматическая обработка опоздавших (перемещение в конец или удаление)
- Панель управления организатора с полным контролем
- Тёмная тема, мобильная адаптация

## Запуск

### Через Docker (рекомендуется)

```bash
docker compose up --build
```

Приложение будет доступно на `http://localhost:8000`.

### Локально (без Docker)

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

## Как пользоваться

### Организатор

1. Перейдите на главную страницу и нажмите "Создать очередь"
2. Укажите название, настройте слоты (время начала, длительность, макс. человек в слоте)
3. Сохраните — получите QR-код и 4-значный код для участников
4. В панели управления:
   - **Вызвать следующего** — вызывает первого в очереди
   - **Обслужён / Пропущен** — отмечает результат
   - **Перемещение** — перетаскивайте участников между слотами
   - **Запросы** — одобряйте или отклоняйте заявки на смену
   - **Настройки** — меняйте параметры на лету

### Участник

1. Отсканируйте QR-код или введите 4-значный код на главной странице
2. Введите имя и присоединитесь к очереди
3. Вы автоматически распределены по слоту
4. При необходимости:
   - Нажмите "Задерживаюсь" если опаздываете
   - Используйте "Сменить слот" или "Позиция в слоте" (если разрешено организатором)

- **Frontend:** Jinja2, vanilla JS, CSS (тема)
